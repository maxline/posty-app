# Welcome!

This is your **first sample post**. It's written in plain Markdown and rendered
by the app at request time.

## Features

- Posts live as `.md` files in the `posts/` folder
- Front matter (`title`, `date`) controls how a post is listed
- New posts can be added through the built-in web form at `/posts/new`
- Markdown syntax like *italics*, **bold**, `code`, and lists all just work
